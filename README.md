# labb_vue

Simple ToDo webbapplication for my second assignment in my frontend course
## Project Setup
```sh
git clone git@github.com:MikaelEdwartz/labb_vue.git
```
 
```sh
cd labb_vue
```

```sh
npm install
```
### Run dev server
```sh
npm run dev
```
### Type-Check, Compile and Minify for Production
```sh
npm run build
```
